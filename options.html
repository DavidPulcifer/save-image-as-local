Save Image As (Local Clean)

Local-only image saver for PNG, JPG, WebP, original files, CSS background images, canvases, and inline SVG.

Output settings

#ffffff

Permissions explained

  • All sites: needed to read the image you right-clicked and to support CSS background images and cross-origin image downloads.
  • Downloads: needed to save the converted file to disk.
  • Storage: only used for these local settings.
  • Offscreen: used to convert images in a hidden extension page without touching the current website.

Notes

  • For file:// pages, enable Allow access to file URLs in chrome://extensions.
  • Original-format downloads preserve the source bytes when possible. Converted JPG / PNG / WebP files are re-encoded.
  • Inline SVG can be saved as SVG or rasterized to PNG / JPG / WebP.
